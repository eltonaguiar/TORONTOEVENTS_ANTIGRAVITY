# Task 09: Audit and Fix Pricing Errors

## Goal
Systematically audit all events for pricing errors and fix them.

## Steps
1. Find the flagged Thursday Bangarang event
2. Verify current price vs expected ($10-$15)
3. Fix the flagged event
4. Audit all Thursday events for pricing issues
5. Audit all other events for common pricing errors
6. Generate correction report

## Success Criteria
- Flagged event price corrected
- All pricing errors identified
- Corrections applied to events.json
- Report generated with all fixes
